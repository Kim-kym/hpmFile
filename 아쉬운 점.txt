S3Service.java

Presigned URL 생성 메서드 

리액트에서 직접 S3로 파일을 업로드 할 수 있도록 
일시적으로 유효한 업로드 전용 URL을 생성해 반환. 
프론트에서 이 URL을 통해 직접 S3로 업로드 가능.    
=> 서버 부하를 줄이고 확장성을 고려.     



통해 프론트엔드에서 직접 S3로 파일을 업로드. 
서버는 파일만 관리하고, 실제 업로드는 사용자가 직접 S3로 업로드. 
 
아쉬웠던 점 

- 기능 구현 미흡 
프론트에서 직접 이미지를 올리는 방식을 
게시판 이미지 업로드에도 구현하고 싶었으나
원인을 제대로 파악하지 못해서 구현하지 못했음
기능 완성은 추후 과제로 남김 

- 다양한 구현 방식에 대한 고민의 여지 부족 
기능 구현에 있어 다양한 방식의 시도가 부족했음
현재 프로젝트에 어떠한 방식이 적절한지, 
하나의 해결책에만 집중하다 보니, 다른 방법들을 살펴볼 여유가 부족했음
추후 더 많은 프로젝트를 진행해보면서 배워나갈 예정.
